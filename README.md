# pharos-skills

Landing page for the two **Pharos AI Agent Carnival — Phase 1** skills.

**Live site:** [hosein-ul.github.io/pharos-skills](https://hosein-ul.github.io/pharos-skills)

## What's here

A single-file dark-themed landing page that:

- Showcases both skills side-by-side
- Documents the 6-check pipeline of `pharos-tx-guardrail`
- Surfaces live TVL data from `pharos-rwa-yield-router`
- Provides install instructions for 4 agent frameworks:
  - Pharos Skill Engine (official)
  - Claude Code
  - Anvita Flow
  - Manual / Any AI Agent
- Lists all verified Pharos mainnet protocol addresses

## Skills featured

| Skill | Purpose | Repo |
|-------|---------|------|
| [pharos-tx-guardrail](https://github.com/hosein-ul/pharos-tx-guardrail) | Pre-execution transaction safety — 6 checks → risk score 0–100 → PROCEED/WARN/BLOCK | `hosein-ul/pharos-tx-guardrail` |
| [pharos-rwa-yield-router](https://github.com/hosein-ul/pharos-rwa-yield-router) | Live RWA yield scanner — reads on-chain APY, risk-adjusted ranking | `hosein-ul/pharos-rwa-yield-router` |

## Tech

Single `index.html`, no build step. Google Fonts only external dependency. Deployed via GitHub Pages.
