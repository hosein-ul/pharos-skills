the bundle

Two skills that compose

The yield router decides where idle capital should go. The tx guardrail verifies that the call to put it there is safe before it is signed.

01 /pharos-tx-guardrail safety

Pre-execution checkpoint. Runs 6 read-only checks against the target contract, calldata, and simulation. Returns a risk score 0–100 with a clear PROCEED / WARN / BLOCK recommendation. Catches unlimited approvals, ownership transfers, contract typos, and value drains before they hit the chain.

// trigger: any cast send / forge --broadcast $ agent → guardrail.check(target=0xE0BE...64ec8, calldata=0x095ea7b3...) › risk = 70/100 HIGH unlimited_approval detected › action = BLOCK — replace MAX_UINT256 with exact amount
6 checks 0 on-chain writes ~120ms per call v0.2.0
02 /pharos-rwa-yield-router intelligence

Read-only yield scanner across Pharos's RWA stack. Reads live APY from R25 Axil, Zona, Morpho Blue, TermMax, AquaFlux, Faroo. Multiplies by asset-class risk factors (0.50 – 0.95), filters by user tolerance and lockup, returns a ranked recommendation.

$ agent → yield.scan(capital=5000 USDC, risk=balanced) › #1 R25 Axil 7-day 8.5% × 0.60 = 5.10% ($892K TVL) › #2 R25 Axil 6-month 11.2% × 0.60 = 6.72% ($6.83M TVL) › #3 Zona USDC supply 3.6% × 0.80 = 2.88% ($620K)
9 protocols tracked 0 on-chain writes $7.7M+ live TVL v0.3.0

onboarding

From zero to first scan in 4 minutes

No build step. No compiler. The skill is markdown + JSON read by your agent at runtime. You just need Foundry's cast on your PATH.

Install Foundry

Both skills run on top of cast (read-only RPC calls). One-line install on Linux/macOS/WSL.

$ curl -L https://foundry.paradigm.xyz | bash && foundryup $ cast --version cast Version: 0.2.0

Clone the bundle

One repo, both skills. Pick your destination path based on your agent (see the install matrix above).

$ git clone https://github.com/hosein-ul/pharos-skills ~/pharos-skills

Verify a live read

Sanity-check the Chainlink PROS/USD price feed on Pharos mainnet — confirms RPC, oracle address, and your local cast install all work.

$ cast call 0x9356C87a48F913d11C87a0d4b8cD16CD04624BF3 \ "latestAnswer()(int256)" --rpc-url https://rpc.pharos.xyz 1240000000000000000 // $1.24 PROS

Run your first guarded transaction

Ask your agent something like "approve 500 USDC to my Zona pool". Watch the guardrail fire before the agent signs.

› agent: // detected: cast send approve(...) › guardrail check #1 simulation ........... PASS › guardrail check #2 contract_exists ...... PASS (USDC, 1798 bytes) › guardrail check #3 selector ............. approve(address,uint256) › guardrail check #4 unlimited_approval ... PASS (exact amount) › guardrail check #5 value_check .......... PASS › guardrail check #6 gas_estimate ......... PASS (71,897 gas) › RISK 5/100 — SAFE — PROCEED
