Onboarding

From zero to first scan, in four soft minutes.

No build step. No compiler. Just markdown your agent reads at runtime.

i

Install Foundry

Both skills shell out to cast for read-only RPC.

curl -L https://foundry.paradigm.xyz | bash foundryup
ii

Clone the bundle

One repo, both skills.

git clone \ hosein-ul/pharos-skills \ ~/pharos-skills
iii

Verify a read

PROS/USD via Chainlink.

cast call 0x9356C87a... \ "latestAnswer()(int256)" \ --rpc-url https://rpc.pharos.xyz
iv

Run guarded tx

Prompt your agent. Watch it think.
