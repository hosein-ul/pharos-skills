The argument is simple. As autonomous coding agents move further into transaction-signing territory on Pharos's mainnet, the gap between "the agent meant well" and "the user just lost their balance" is one keystroke wide. These two skills close that gap from both sides — the safety side, with a deterministic checklist; and the yield side, with honest, on-chain reads instead of marketing APR.

The first skill, pharos-tx-guardrail, runs six read-only checks against any pending transaction. It catches unlimited approvals, ownership transfers, and missing contract code before they ever reach the signer. The second, pharos-rwa-yield-router, reads live APY from nine protocols, applies asset-class risk multipliers between 0.50 and 0.95, and returns a ranked recommendation.

Both skills ship as single-folder Pharos Skill Engine packages — markdown plus JSON, no build step, no compiler. They are designed to be hosted by any agent runtime that can read a SKILL.md file. Install instructions for eleven such runtimes appear in the directory below.

The Skills

Two essays in safer software.

One skill decides where idle capital should go. The other verifies the call to put it there is safe. Together they cover the full lifecycle of a Pharos transaction.

SKILL · 01

pharos-tx-guardrail

A pre-execution checkpoint.

Runs six read-only checks against the target contract, calldata, and simulation. Returns a risk score between zero and one hundred, with a clear PROCEED / WARN / BLOCK recommendation.

Onboarding · 4 Chapters

From zero to first scan, in four minutes.

No build step. No compiler. The skill is markdown and JSON your agent reads at runtime. You only need Foundry's cast on your PATH.

i.

Install Foundry

Both skills shell out to cast for read-only RPC calls. One line on Linux, macOS, or WSL.

curl -L https://foundry.paradigm.xyz | bash foundryup cast --version
ii.

Clone the bundle

One repository, both skills. Pick the destination path that matches your agent — see the directory above.

git clone https://github.com/hosein-ul/pharos-skills \ ~/pharos-skills
iii.

Verify a live read

Confirm RPC, oracle address, and your local cast install all work, via the Chainlink PROS/USD feed.

cast call 0x9356C87a48F913d11C87a0d4b8cD16CD04624BF3 \ "latestAnswer()(int256)" --rpc-url https://rpc.pharos.xyz # 1240000000000000000 → $1.24 PROS
iv.

Run a guarded transaction

Ask your agent something Pharos-related. Watch the guardrail fire before it signs.

› cast send approve(...) › check #4 unlimited_approval .. PASS › RISK 5/100 — SAFE — PROCEED ✓
