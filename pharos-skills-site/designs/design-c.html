Onboarding

Zero to first scan
in four minutes.

No build step. No compiler. The skill is markdown + JSON read by your agent at runtime. You only need Foundry's cast on your PATH.

STEP 01 · ONE MINUTE

Install Foundry

Both skills shell out to cast for read-only RPC calls. One-line install on Linux / macOS / WSL.

curl -L https://foundry.paradigm.xyz | bash && foundryup cast --version # cast Version: 0.2.0
STEP 02 · 30 SECONDS

Clone the bundle

One repo, both skills. Pick the destination path that matches your agent (see the install matrix above).

git clone https://github.com/hosein-ul/pharos-skills ~/pharos-skills
STEP 03 · 30 SECONDS

Verify a live read

Sanity-check the Chainlink PROS/USD price feed on Pharos mainnet — confirms RPC, oracle address, and your local cast install all work.

cast call 0x9356C87a48F913d11C87a0d4b8cD16CD04624BF3 \ "latestAnswer()(int256)" --rpc-url https://rpc.pharos.xyz # 1240000000000000000 → $1.24 PROS
STEP 04 · TWO MINUTES

Run your first guarded transaction

Prompt your agent with something Pharos-related: "approve 500 USDC to my Zona pool" — and watch the guardrail fire before the agent signs.
