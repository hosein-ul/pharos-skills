Agent Installation & Setup Workflow

1. Clone Repos

The agent clones both repositories into the platform's local skills directory (e.g. Cursor rules or Claude Code skills).

2. Bootstrap

Agent checks for cast, python, and curl on the system and installs cast quietly if missing.

3. Hot Wallet

If AGENT_PRIVATE_KEY is absent, agent generates a new wallet, stores it in .env, and asks you to fund it.

4. Ready to Go

Agent scans all 3 skills, presents capabilities in plain English, and waits for your confirmation to send transactions.
